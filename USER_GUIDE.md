# User Guide

```text
RK TELEPROMPTER PRO V3 — SMART CONNECT EDITION
=================================================

FILES
-----
1. RK_Teleprompter_Pro_V3_SmartConnect.html
   The complete offline teleprompter in one file.

HOW TO USE ON THE LENOVO TABLET
-------------------------------
1. Transfer the HTML file to the tablet using Lenovo Smart Connect, USB, Google Drive,
   email, Nearby Share/Quick Share, or another trusted method.
2. Open the file using Chrome, Edge, or another Chromium-based browser.
3. Paste or import a script.
4. Adjust mirror mode, font size, text width, reading-line position, and pace.
5. Press Start Prompt.
6. Click/tap once inside the prompt when using laptop keyboard control through Smart Connect.
7. Press H to hide the controls and HUD for a clean reflection.

DEFAULT CONTROLS
----------------
Space                 Play/Pause
Up / Down             Faster/Slower
Left / Right           Nudge backward/forward
Page Up / Page Down    Previous/Next named mark
+ / -                  Increase/Decrease font size
M                      Toggle horizontal mirror
G                      Toggle guide band
H                      Toggle HUD and controls
R                      Restart
Home / End             Beginning/End
N                      Take +1
Esc                    Exit prompt
Mouse wheel            Manually nudge script
Shift + mouse wheel    Adjust pace

TOUCH GESTURES
--------------
Tap                    Play/Pause
Swipe left/right       Previous/Next mark
Swipe up/down          Faster/Slower

MARKERS
-------
<<mark:INTRO>>         Named navigation point
<<pause:2>>            Timed two-second pause
<<pause>>              Pause until manually resumed
<<speed:70>>           Change to 70 pixels per second
<<wpm:135>>            Change to 135 words per minute
<<cue:SMILE>>          Temporary on-screen performance cue
<<countdown:3>>        Mid-script countdown

BACKUP
------
Use “Backup JSON” for the current script or “Export all scripts” for a complete
library/settings backup. Keep a copy outside the tablet before clearing browser data.

NOTES
-----
- The app itself does not upload scripts or analytics.
- Voice recognition, wake lock, orientation lock, and some fullscreen behaviours depend
  on browser/device permissions. They may work more reliably from HTTPS hosting or an APK.
- The web app can darken its own screen but cannot change Android hardware brightness.
- Smart Connect does not require a custom integration: keyboard, mouse, touchpad,
  clipboard and screen-control inputs are handled by the web app.

```
