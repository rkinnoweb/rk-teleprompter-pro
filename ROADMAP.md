# Roadmap

## Recommended next improvements

### Near term
- Add paragraph-focus mode with current paragraph emphasis.
- Add rehearsal analytics stored locally: elapsed time, pauses, manual speed changes and completed takes.
- Add an optional safe-zone preview for different teleprompter glass sizes.
- Add a release builder that regenerates the single-file HTML from the multi-file source.
- Add browser-level smoke tests for core controls and marker handling.

### Later
- Progressive Web App packaging for home-screen installation and offline caching.
- Optional two-device controller over the local network, with no cloud dependency.
- Native Android wrapper for stronger brightness, orientation and Bluetooth-key handling.
- Script formatting tools for breath breaks, emphasis and multilingual typography.

## Decision rule
Prioritize reliability, data safety and hands-free control over decorative features.
